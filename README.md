# Capstone_1
Capstone 1: Business Analysis with SQL and Excel

This Capstone will show our skills on how we can assemble shape and analyze data for a fictional business. The skills used for this project are within SQL and Microsoft Excel. 

The following is included in this project,
One SQL script showing the generated results of the sales territory report.
At least one Excel PivotChart illustrating one of the analysis questions.
A short description and overview of the project.
A 10 minute video recording to demonstrate the project and results.
